## Development Lifecycle.
Development lifecycle adalah seluruh proses pengembangan dari sebuah software. Untuk Mengembangkan sebuah project tidak cukup hanya melakukan coding. Banyak tahapan.
1. Menetukan requirements bisnis yang diperlukan.
2. Menentukan Desain arsitektur dan Teknologi.
3. Eksekusi Pengerjaan dengan `ngoding`.
4. Membuat Dokumentasi dari sebuah project yang dikerjakan.
5. mengatur distribusi project.

## Brief Umum.
1. Setiap sebuah web pasti memiliki 2 bagian besar. Kita Menyebutnya :
    - `frontend` 
    - `Backend`
2. Setiap web memiliki satu set teknologi yang digunakan. Kita Menyebutnya `Stack Technology`.
